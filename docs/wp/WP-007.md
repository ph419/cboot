## 问题分析

### 错误现象

旧版本 cboot (pre-v1.0.2) 创建的 settings 文件不包含 `CLAUDE_CODE_DISABLE_1M_CONTEXT` 和 `CLAUDE_CODE_AUTO_COMPACT_WINDOW` 属性。当用户升级到最新版 cboot 并尝试修改上下文窗口配置时，代码直接对 `PSCustomObject` 赋值不存在的属性，导致异常。

**错误信息**：
```
发生错误: 设置"CLAUDE_CODE_DISABLE_1M_CONTEXT"时发生异常:"在此对象上找不到属性"CLAUDE_CODE_DISABLE_1M_CONTEXT"。请确认该属性存在并且可设置。"
```

### 根本原因

`ConvertFrom-Json` 反序列化的 `PSCustomObject` 对象，其属性集合在创建时固定。当 settings 文件来自旧版本（不包含上下文窗口相关字段）时，直接赋值新属性会抛出异常。

### 影响范围

| 函数 | 行号 | 影响 |
|------|------|------|
| `Add-Model` | 1263 | 静默失败，用户无感知但配置丢失 |
| `Add-Model` | 1268 | 未检查直接 Remove，可能静默失败 |
| `Edit-ModelConfig` | 1615 | 抛出异常，用户可见错误 |

## 实施计划

### Step 1: 修复 Add-Model 函数 (行 1263-1265)

**位置**: `cboot.ps1` 第 1263-1265 行

**当前代码**：
```powershell
$settings.env.CLAUDE_CODE_DISABLE_1M_CONTEXT = "1"
$compactWindow = $contextWindowSize
$settings.env.CLAUDE_CODE_AUTO_COMPACT_WINDOW = "$compactWindow"
```

**修复为**：
```powershell
$settings.env | Add-Member -MemberType NoteProperty -Name 'CLAUDE_CODE_DISABLE_1M_CONTEXT' -Value "1" -Force
$compactWindow = $contextWindowSize
$settings.env | Add-Member -MemberType NoteProperty -Name 'CLAUDE_CODE_AUTO_COMPACT_WINDOW' -Value "$compactWindow" -Force
```

**说明**: 使用 `Add-Member -Force` 可以安全地添加或更新属性，无论属性是否存在。

### Step 2: 修复 Add-Model 函数 (行 1268-1269)

**位置**: `cboot.ps1` 第 1268-1269 行

**当前代码**：
```powershell
$settings.env.PSObject.Properties.Remove('CLAUDE_CODE_DISABLE_1M_CONTEXT') | Out-Null
$settings.env.PSObject.Properties.Remove('CLAUDE_CODE_AUTO_COMPACT_WINDOW') | Out-Null
```

**修复为**：
```powershell
if ($settings.env.PSObject.Properties['CLAUDE_CODE_DISABLE_1M_CONTEXT']) {
    $settings.env.PSObject.Properties.Remove('CLAUDE_CODE_DISABLE_1M_CONTEXT')
}
if ($settings.env.PSObject.Properties['CLAUDE_CODE_AUTO_COMPACT_WINDOW']) {
    $settings.env.PSObject.Properties.Remove('CLAUDE_CODE_AUTO_COMPACT_WINDOW')
}
```

**说明**: 与 `Edit-ModelConfig` 函数（行 1620-1625）保持一致的防御性编程模式。

### Step 3: 修复 Edit-ModelConfig 函数 (行 1615-1617)

**位置**: `cboot.ps1` 第 1615-1617 行

**当前代码**：
```powershell
$settings.env.CLAUDE_CODE_DISABLE_1M_CONTEXT = "1"
$compactWindow = $newContextSize
$settings.env.CLAUDE_CODE_AUTO_COMPACT_WINDOW = "$compactWindow"
```

**修复为**：
```powershell
$settings.env | Add-Member -MemberType NoteProperty -Name 'CLAUDE_CODE_DISABLE_1M_CONTEXT' -Value "1" -Force
$compactWindow = $newContextSize
$settings.env | Add-Member -MemberType NoteProperty -Name 'CLAUDE_CODE_AUTO_COMPACT_WINDOW' -Value "$compactWindow" -Force
```

**说明**: 同 Step 1，使用 `Add-Member -Force` 确保属性安全赋值。
