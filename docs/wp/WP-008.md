## 目标

将所有 statusLine 配置从 `ccsp --preset MBTC --theme powerline` 迁移到 `ccstatusline`，同时新增 `refreshInterval: 10` 属性。PostToolUse hook 中的 ccsp 命令也一并更新。

## 实施计划

### Step 1: 更新 config/settings/ 下 4 个模板文件

文件列表：
- `config/settings/settings-glm.example.json` (第 93-96 行)
- `config/settings/settings-glm-5-turbo.example.json` (第 93-96 行)
- `config/settings/settings-glm-5v-turbo.example.json` (第 93-96 行)
- `config/settings/settings-glm-5.1.example.json` (第 93-96 行)

替换：
```json
// 旧
"statusLine": {
    "type": "command",
    "command": "ccsp --preset MBTC --theme powerline"
  }

// 新
"statusLine": {
    "type": "command",
    "command": "ccstatusline",
    "refreshInterval": 10
  }
```

### Step 2: 更新 cboot.ps1 模板中的 statusLine（第 277-280 行）

```powershell
# 旧
statusLine = @{
    type = "command"
    command = "ccsp --preset MBTC --theme powerline"
}

# 新
statusLine = @{
    type = "command"
    command = "ccstatusline"
    refreshInterval = 10
}
```

### Step 3: 更新 cboot.ps1 PostToolUse hook 命令（第 233 行）

```powershell
# 旧
command = "ccsp --preset MBTC --theme powerline"

# 新
command = "ccstatusline"
```
